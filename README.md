### Directory tree structure written in Rust
list of all file and folders in tree structure

### Installation
```shell
$ cargo install rtree
```

### Usage

```shell
$ rtree ./

Path: ./
  ▕─ dir2
  ▕   ▕─ inner dir 1
  ▕   ▕    ▕─ file1
  ▕   ▕    ▕─ file2
  ▕─ dir1
  ▕   ▕─ file1
  ▕   ▕─ file2
  ▕─ dir3
```
